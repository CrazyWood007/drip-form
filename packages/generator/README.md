# @drip/form-generator
用于Drip-form的表单生成器

## 技术选型

- 状态管理：[recoil](https://recoiljs.org/)
- 拖拽渲染：[dnd-kit](https://dndkit.com/)
- 组件库：[drip-design](http://drip-design.jd.com/docs/spec/introduce-cn)
- 样式库：[tailwindcss](https://www.tailwindcss.cn/)

## 编辑器插件（建议安装）
- postcss
- tailwindcss
